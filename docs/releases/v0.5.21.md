# Release Notes — v0.5.21

**Date:** 2026-01-21
**Repository:** agentic-career-search

## Highlights

This release focuses on stability and developer-experience improvements
to the `autoapply_agent` core.

## Changes

### Added
- `resume` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed agent trace output
- Unit tests for crawler boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running resume worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average agent latency by ~12% via connection pooling
- Cleaner error messages for crawler validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade agentic-career-search
```
