# Release Notes — v0.1.14

**Date:** 2024-08-12
**Repository:** agentic-career-search

## Highlights

This release focuses on stability and developer-experience improvements
to the `autoapply_agent` core.

## Changes

### Added
- `workflow` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed resume trace output
- Unit tests for job boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running workflow worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average resume latency by ~12% via connection pooling
- Cleaner error messages for job validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade agentic-career-search
```
